# Task Manager built using React

## Using JSON Server to host the tasks

## In root:

```bash
npm install
```

## To build:

```bash
npm run build
```

## For JSON Server Database(port 5000)

```bash
npm run server
```

## Install 'serve' npm package, run server on chosen port

```bash
npm i -g serve

serve -s build -p ####
```
